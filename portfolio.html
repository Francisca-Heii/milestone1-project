Distributed system

Managing day to day task in a library can be a tedious and time-consuming job. A Library Management System has been proposed to facilitate and easily maintain day to day library tasks. An Admin, Librarian, Teachers and Students will have access to the LMS. The system is capable of handling requests for issuing, renewing and returning books by both teachers and students.

Technology

MySQL has been used to create and implement the database. PHP was used to model the overall system. AJAX was used to facilitate application server communication and HTML/CSS was used for styling the web pages.

Online food delivey

The application can be used by restaurant owners to register themselves and then view and edit their menus and manage incoming orders. Customers can use the application to view the restaurants, and place and track their orders and do payment online. Additionally, Open Street Maps have been included so that customers can view the locations of the restaurants.

Technology

PHP for the backend and MySQL as the Data Base Management System and Bootstrap with HTML/CSS as the front end.
